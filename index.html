Download

You can download this project in either zip or tar formats:

  • Core: zip tar
  • Persistent List Manager: zip tar/li>
  • Registration: zip tar/li>

You can also clone any of these projects with Git by running:

  • Core: 
    $ git clone git://github.com/OpenHMIS/core
  • Persistent List Manager: 
    $ git clone git://github.com/OpenHMIS/plm
  • Registration: 
    $ git clone git://github.com/OpenHMIS/registration
